# MindNote 지원 (Support)

MindNote는 하루 한 줄로 마음을 기록하고, AI가 감정을 분석해주는 감정 일기 iOS 앱입니다.

## 자주 묻는 질문 (FAQ)

### 기록은 어디에 저장되나요?

모든 기록은 사용자의 기기에만 저장됩니다. 다만 AI 감정 분석을 직접 요청한 경우에만 해당 문장이 분석을 위해 일시적으로 외부 AI에 전송되며, 저장되지 않습니다.

### 계정이 필요한가요?

아니요. 계정 생성이나 로그인 없이 바로 사용할 수 있습니다.

### 기록을 백업할 수 있나요?

설정에서 JSON 파일로 내보내기/불러오기를 지원합니다. 앱을 삭제하면 기기의 기록도 함께 삭제되니 중요한 내용은 미리 백업해 주세요.
